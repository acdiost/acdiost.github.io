  • 培训经历

    Oct 14, 2025

    今天面试了一个培训的人，26 岁，他是达内培训机构出来的，在达内交了近两万的培训费用。以前是做的贷款销售，一个月 4 千左右在成都，我仿佛看到了当年的自己。

    回想 9 年前，我也经历了 Java 培训，交了 16800 块，分期。

    当时的境遇特别需要一份入门的工作，又没有什么存款，算是比较窘迫了吧。

    现在看着一份份投递过来的培训简历，很明显的简历包装，想到了自己以前的过往，心想还是给他们一个面试机会，让他们试试吧，说不定能改变某人的命运呢。

    还有一个尴尬的情况，公司需要来了能干活的人，因为公司也需要生存，暂时无法留有余地去培养经验不丰富的人。经验不丰富的人又需要工作机会来积累经验，形成了冲突。

    站在不同的立场上，也因为他的面试情况不佳，我没有通过他的面试。做出这个决定时，我心里还是有些遗憾的，我理解他的焦虑，也理解公司的现实——但现实终究不是怜悯能改变的。

    我想机会还是要给内心真正喜欢技术和这个行业的人，而不是谋求生存而进入该行业的人。

    也许每一次面试，都是一场命运交叉的瞬间。有人求生，有人求梦，而我，只希望能遇见那个为热爱而坚持的人。

    波粒二象性

    什么是波粒二象性？

    波粒二象性（wave-particle duality）指的是所有的粒子或量子不仅可以部分地以粒子的术语来描述，也可以部分地用波的术语来描述。【百科】

    波粒二象性是指微观粒子（如光子、电子等）在某些实验条件下既表现出波动性质又表现出粒子性质的现象。这是量子力学的一个基本概念，最早由德国物理学家德布罗意（Louis de Broglie）于1924年提出。【ChatGPT】

    个人不懂相关的知识，粗鄙的愚见：

    它类比就像水和水分子。

    波即是水，粒即是水分子。

    当达到一定质量即为波，仔细观察即是水分子。

    推测其甚至可能还有固态形式。
