  • 写了一个 hugo 主题

    Feb 8, 2024

    在上个月末，我突发奇想，决定自己结合 ChatGPT 写一个 Hugo 主题。 最开始通过网上的几个视频教程学习，如何创建主题开始一个 Hello world， 再参考其他主题的语法，把基础的几个页面架子搭建出来了。

    然后开始学习 Tailwindcss，也间接了解了 Unocss。

    最后通过和 ChatGPT 的对话，写了三遍，我成功地完成了此主题编写。

    期间因为工作、其他事情的耽搁，花费了将近半个月才把这个主题完全写好。
